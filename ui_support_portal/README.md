# BookLeaf Help Desk

Author support and communication portal for BookLeaf Publishing.

## Tech stack

- Vite
- JavaScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Build

```sh
npm run build
```

## .env

```env
VITE_API_URL=http://localhost:8000
```
